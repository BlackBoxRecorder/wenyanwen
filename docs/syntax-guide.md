# 文言文标记语法使用说明

本文档介绍 `.wyw` 文件的标记语法，用于编写带注释、注音和现代文翻译的文言文。

---

## 文件结构

`.wyw` 文件由两部分组成：
1. **Frontmatter** - 文件顶部的元数据（可选）
2. **正文内容** - 实际的文言文及标记

---

## 一、Frontmatter（元数据）

文件开头使用 YAML 格式定义文档元数据，包裹在 `---` 之间：

```wyw
---
title: 陋室铭
author: 刘禹锡
dynasty: 唐
source: 全唐文
layout: ancient
---
```

### 支持的字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 文章标题 | `陋室铭` |
| `author` | 作者姓名 | `刘禹锡` |
| `dynasty` | 所属朝代 | `唐` |
| `source` | 出处来源 | `全唐文` |
| `layout` | 布局样式 | `ancient`（古典样式） |

---

## 二、块级语法

### 1. 标题

使用 `#` 表示标题，支持 1-3 级：

```wyw
# 一级标题
## 二级标题
### 三级标题
```

### 2. 段落

普通文本行自动识别为段落。段落之间用空行分隔：

```wyw
这是第一段文言文。

这是第二段文言文。
```

### 3. 译文

使用 `>>` 标记现代文翻译，紧跟在对应文言文段落后：

```wyw
山不在高，有仙则名。

>> 山不在于有多高，有了仙人居住就会出名。
```

**特点：**
- 译文会自动与上方段落关联
- 支持多行译文（每行都以 `>>` 开头）

### 4. 引用块

使用 `>` 标记引用内容：

```wyw
> 孔子曰：学而时习之，不亦说乎？
```

### 5. 分隔线

使用三个或以上连字符表示分隔线：

```wyw
---
```

### 6. 诗词围栏块

使用 `:::` 包裹诗词内容，支持内部标题和元信息：

```wyw
::: poetry
# 静夜思
:: 李白

床前明月光，
疑是地上霜。

举头望明月，
低头思故乡。
:::
```

**说明：**
- `::: poetry` - 开始围栏块（类型默认为 poetry）
- `# 静夜思` - 诗词标题
- `:: 李白` - 作者元信息
- `:::` - 结束围栏块

## 三、内联语法

### 1. 注音（Ruby 标注）

使用 `{字|拼音}` 为汉字添加拼音标注：

```wyw
有{仙|xiān}则{名|míng}
```

**效果：** 在"仙"字上方显示拼音 "xiān"

### 2. 注释

使用 `[词](释义)` 为词语添加注释：

```wyw
[陋室](简陋的屋子)，惟吾[德馨](品德高尚)。
```

**效果：** "陋室"显示为带注释的词，释义为"简陋的屋子"

### 3. 着重（强调）

使用 `*文本*` 标记需要强调的内容：

```wyw
*孔子*云：何陋之有？
```

### 4. 专名标记

使用 `_文本_` 标记专有名词（人名、地名等）：

```wyw
南阳_诸葛庐_，西蜀_子云亭_。
```

**效果：** "诸葛庐"和"子云亭"以专名样式显示

### 5. 书名标记

使用 `《书名》` 标记书籍名称：

```wyw
阅《金经》，调《素琴》。
```

## 四、完整示例

```wyw
---
title: 陋室铭
author: 刘禹锡
dynasty: 唐
source: 全唐文
layout: ancient
---

山不在高，有{仙|xiān}则名。水不在深，有{龙|lóng}则{灵|líng}。[斯](这)是[陋室](简陋的屋子)，惟吾[德馨](品德高尚)。

>> 山不在于有多高，有了仙人居住就会出名。水不在于有多深，有了龙的存在就会有灵气。这是一间简陋的屋子，只因我的品德好就不感到简陋了。

[苔](青苔)痕上阶绿，草色入[帘](遮蔽门窗的用具)青。谈笑有[鸿儒](博学的人)，往来无[白丁](平民，指没有学问的人)。

>> 苔藓碧绿，长到台阶上；草色青翠，映入竹帘里。到这里谈笑的都是博学之人，交往的没有知识浅薄之人。

可以调{素琴|sù qín}，阅{金经|jīn jīng}。无[丝竹](琴瑟、箫管等乐器的总称)之乱耳，无[案牍](官府的公文)之[劳](使……劳累)形。

>> 可以弹奏素朴的古琴，阅读珍贵的佛经。没有世俗的乐声扰乱心境，没有官府的公文劳累身心。

南阳_诸葛庐_，西蜀_子云亭_。*孔子*云：何陋之有？

>> 南阳有诸葛亮的草庐，西蜀有扬子云的亭子。孔子说："这有什么简陋的呢？"
```

## 五、语法速查表

| 语法 | 用途 | 示例 |
|------|------|------|
| `---` | Frontmatter 分隔 | `---\ntitle: 标题\n---` |
| `#` `##` `###` | 标题 | `# 一级标题` |
| `>>` | 译文 | `>> 现代文翻译` |
| `>` | 引用 | `> 引用内容` |
| `---` | 分隔线 | `---` |
| `:::` | 围栏块 | `::: poetry\n内容\n:::` |
| `{字\|拼音}` | 注音 | `{仙\|xiān}` |
| `[词](释义)` | 注释 | `[陋室](简陋的屋子)` |
| `*文本*` | 着重 | `*强调*` |
| `_文本_` | 专名 | `_诸葛庐_` |
| `《书名》` | 书名 | `《论语》` |

---

## 六、注意事项

1. **空行分隔**：段落之间需要空行分隔，否则会被合并为一个段落
2. **译文关联**：译文块（`>>`）会自动与紧邻的上方段落关联
3. **嵌套支持**：着重标记 `*` 和专名标记 `_` 支持内部嵌套其他内联语法
4. **转义**：目前暂不支持转义，避免在文本中直接使用特殊标记字符
